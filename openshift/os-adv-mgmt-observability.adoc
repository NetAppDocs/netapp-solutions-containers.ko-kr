= 관찰 가능성
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Kubernetes를 위한 고급 클러스터 관리 기능은 모든 클러스터에서 노드, 포드, 애플리케이션 및 워크로드를 모니터링하는 방법을 제공합니다.

. 환경 관찰 > 개요로 이동합니다.
+
image:redhat-openshift-082.png["관찰 가능성 홈페이지"]

. 모든 클러스터의 모든 포드와 워크로드는 다양한 필터를 기준으로 모니터링되고 정렬됩니다.  해당 데이터를 보려면 Pod를 클릭하세요.
+
image:redhat-openshift-083.png["포드 관찰"]

. 다양한 데이터 포인트를 기반으로 클러스터 전반의 모든 노드를 모니터링하고 분석합니다.  노드를 클릭하면 해당 세부 정보에 대한 자세한 정보를 얻을 수 있습니다.
+
image:redhat-openshift-084.png["노드 관찰"]

. 모든 클러스터는 다양한 클러스터 리소스와 매개변수를 기반으로 모니터링되고 구성됩니다.  클러스터 세부 정보를 보려면 클러스터를 클릭하세요.
+
image:redhat-openshift-085.png["클러스터 관찰"]
